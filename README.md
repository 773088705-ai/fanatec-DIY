# fanatec-DIY
基于3.3vPro micro和霍尔开关模块的fanatec手排挡
可以切换3种模式分别为 
1用USB连接的PC模式，
2用rj12连接基座的H档模式，
3用rj12连接基座的序列档模式。
仅用csw2.5测试，其他基座暂未测试。
我测试过直接用开发版的pwm输出对应的电压，但是效果不是很稳定，依然采用分压电阻网络，经过测试，这是最稳定的
排挡只有6个档位槽，通过按压挡头的按钮来实现7＋R，8个档位
至于机械部分可以自行DIY，我用了一根3030欧标的型材作为底座，一个拖拉机的拨叉轴，直线轴承，硬光轴，联轴器，鱼眼轴承，定位柱塞，以及手动加工的不锈钢板

开发环境为Arduino IDE
